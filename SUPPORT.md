# Support

Thanks for using KavachOS.

## Where to get help

- Usage questions and integration help:
  - GitHub Discussions (preferred)
  - Discord: https://discord.gg/kavachos
- Bugs and reproducible defects:
  - Open an issue using the bug template
- Security vulnerabilities:
  - Email security@kavachos.com
  - Do not open public issues for security reports

## Before opening an issue

Please include:

1. Package name and version (`npm list kavachos` or package.json)
2. Runtime details (Node version, OS, database)
3. Minimal reproduction (repo link or code snippet)
4. Expected behavior vs actual behavior
5. Logs or stack traces with secrets removed

## Response expectations

- Triage acknowledgment target: within 2 business days
- Security reports: see SECURITY.md response SLAs
- Feature requests: reviewed during roadmap planning

## Scope and priorities

Highest priority:

- Security and auth bypass issues
- Data loss and reliability bugs
- Regressions in core auth/agent primitives

Lower priority:

- Requests without reproductions
- Environment-specific setup questions already covered in docs

## Commercial and enterprise support

For enterprise support, contact hello@kavachos.com.
